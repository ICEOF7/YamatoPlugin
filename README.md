# YamatoPlugin

Minecraft Yamato sword plugin for Spigot, Paper and Purpur 1.20.5–1.21.x

---

**YamatoPlugin** добавляет на сервер меч **Yamato**. Это быстрый клинок с активными способностями, зарядом ульты и настраиваемыми сообщениями

  

Плагин подойдёт для PvP, RPG сборок, приватных серверов и мини игр, где нужен красивый кастомный предмет без лишней тяжести


---

## Что есть

  

* кастомный меч Yamato

* несколько активных способностей

* заряд ульты за убийства

* прогресс заряда в описании предмета

* настройка урона, кулдаунов, предмета и сообщений через `config.yml`

  ---

## Команды

  

```text

/yamato give

/yamato give <ник>

/yamato reload

```

  

Право для команд:

  

```text

yamato.admin

```

  

По умолчанию оно есть у операторов сервера

---

## Способности

  

**ПКМ:** быстрый разрез по цели впереди

  

**Shift и ПКМ:** рывок вперёд с уроном по врагам

  

**F:** стан и удар по области

  

**Shift, прыжок и ЛКМ:** ульта, если заряд уже готов

  

Ульта заряжается за убийства с Yamato. Количество убийств можно поменять в конфиге

---  

## Установка


1. Скачай `jar` файл из релиза

2. Положи его в папку `plugins`

3. Запусти сервер

4. Настрой `plugins/YamatoPlugin/config.yml`

5. Выполни `/yamato reload` или перезапусти сервер


---

## Настройка

  

Главные параметры лежат в `config.yml`. Там можно поменять название меча, материал, описание, сообщения, урон, радиусы, кулдауны и заряд ульты

  

Описание предмета сделано списком строк. Одна строка в конфиге значит одна строка в описании предмета

  

В тексте можно использовать цвета вроде `<red>`, `<dark_purple>`, `<#7c5cff>` и цветовые коды через `&`
